## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| background.ts → storage | Learned and custom rules read from local storage — attacker with extension storage access could inject custom rules |
| background.ts → filename | Download filename from Chrome API is external input — used only for substring match (includes), not eval |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | storageCustomRules matchText | accept | matchText used only as toLowerCase().includes() substring — no regex eval, no code execution |
| T-03-02 | Information Disclosure | storageConflict | accept | pendingConflict stored in local storage, scoped to extension origin — no external exposure |
| T-03-SC | Tampering | npm installs (tailwindcss, lucide-react) | mitigate | Both are well-established packages on npm; verify package names match exactly before install |
